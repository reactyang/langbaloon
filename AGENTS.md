 a game to learn Chinese vocabulary
 play decide how many words to pratice
 player can select words to pratice or random words by system
 words flying as balloon, until all shot down
 new word come to fly say in 3s, how long new word show can be config later.
 player type pinyin to shoot the balloon 
 win the round of  game if all balloon shot down

 use a dictionary(https://www.npmjs.com/package/chinese-lexicon?activeTab=readme) to get words

 English for usr interface(like buttons etc)

 Single page application using react.js
